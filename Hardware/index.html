Hardware

Here's a list of some of the hardware i got and played around with. Including instructions on what they are and how to set them up. Many of them are already outdated, sold or given away for free. However, it might be interesting for some to read about them.

  • Carambola - A MIPS board with Wifi and Network Capabilities
  • Cubox i4 Pro - An IMX6 based ARM Board
  • eCafé - A small ARM based Netbook
  • GGMM E2 Speaker - An AirPlay compatible and Linux based WiFi-Speaker
  • GL-MT300N - A small OpenWRT mainline WiFi Router with two ethernet ports
  • Google Pixel 5 - A Google Pixel Smartphone codenamed "Redfin"
  • Netgear R7800 - A nice Router that is fully supported by OpenWRT
  • Nokia N900 - The best Linux Phone every made
  • Oculus Quest - An android based VR headset
  • Odroid X - An Exynos based ARM development Board
  • Pandaboard - An OMAP4 based ARM Develoment Board
  • Raspberry Pi - First Revision of the Raspberry Pi
  • Sony Xperia XZ1 Compact - Just another android phone
  • Sony Xperia Z1 Compact - Just another android phone
  • TP-Link WDR4300 - A fully OpenWRT supported router
  • Trancent WiFi SD-Card - A hackable WiFi SD Card
