pfDebug

A little tool to monitor the memory usage, peaks and garbage collection.

Usage

- Left-Click on the Bar to reset the Peak - Right-Click on the Frame to open the Analyzer

Installation

1. Download Latest Version 2. Unpack the Zip file 3. Rename the folder "pfDebug-master" to "pfDebug" 4. Copy "pfDebug" into Wow-Directory\Interface\AddOns 5. Restart Wow
