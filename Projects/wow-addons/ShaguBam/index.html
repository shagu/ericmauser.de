ShaguBam

Everytime a new crit record has be done, this addon shows up the new record and plays a bruce lee soundfile. It can be configured to notify via emotes, group chat, raid chat, say, yell or siltently for the player only. By default it will use emotes. To configure, please type: /bam or /shagubam

Installation

1. Download Latest Version 2. Unpack the Zip file 3. Rename the folder "ShaguBam-master" to "ShaguBam" 4. Copy "ShaguBam" into Wow-Directory\Interface\AddOns 5. Restart Wow
