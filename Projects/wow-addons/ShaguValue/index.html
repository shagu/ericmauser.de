ShaguValue

A Vanilla Addon to display item sell and buy values on tooltips using an integrated database. (Hold shift key for extended values)

Notice: This Addon or a superior and maintained version of it, is already included in pfUI. If using this in combination with pfUI you might get bad results or up to performance regressions.

Installation

1. Download Latest Version 2. Unpack the Zip file 3. Rename the folder "ShaguValue-master" to "ShaguValue" 4. Copy "ShaguValue" into Wow-Directory\Interface\AddOns 5. Restart Wow
