pfUI Elite Overlay

An extension for pfUI which adds dragon textures to elite, rare and worldbosses.

Installation (Vanilla, 1.12)

1. Download Latest Version 2. Unpack the Zip file 3. Rename the folder "pfUI-eliteoverlay-master" to "pfUI-eliteoverlay" 4. Copy "pfUI-eliteoverlay" into Wow-Directory\Interface\AddOns 5. Restart Wow

Installation (The Burning Crusade, 2.4.3)

1. Download Latest Version 2. Unpack the Zip file 3. Rename the folder "pfUI-eliteoverlay-master" to "pfUI-eliteoverlay-tbc" 4. Copy "pfUI-eliteoverlay-tbc" into Wow-Directory\Interface\AddOns 5. Restart Wow
