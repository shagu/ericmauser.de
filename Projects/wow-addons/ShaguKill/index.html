ShaguKill

Displays the count of remaining kills or other Exp-giving events that are required to gain the next levelup.

Installation

1. Download Latest Version 2. Unpack the Zip file 3. Rename the folder "ShaguKill-master" to "ShaguKill" 4. Copy "ShaguKill" into Wow-Directory\Interface\AddOns 5. Restart Wow
