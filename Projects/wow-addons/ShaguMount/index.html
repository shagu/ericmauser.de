ShaguMount

A Vanilla Addon that automatically unmounts the player when using an action. It is quite similar to ezDismount, as it unmounts the Player whenever a action is used which cannot be done mounted or shapeshifted, however tt's written minimal and designed to work on every localized gameclient.

Notice: This Addon or a superior and maintained version of it, is already included in pfUI. If using this in combination with pfUI you might get bad results or up to performance regressions.

Installation

1. Download Latest Version 2. Unpack the Zip file 3. Rename the folder "ShaguMount-master" to "ShaguMount" 4. Copy "ShaguMount" into Wow-Directory\Interface\AddOns 5. Restart Wow
