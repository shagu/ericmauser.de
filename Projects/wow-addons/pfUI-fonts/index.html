pfUI-fonts

A font package for pfUI, providing additional fonts from the google font project.

Installation (Vanilla, 1.12)

1. Download Latest Version 2. Unpack the Zip file 3. Rename the folder "pfUI-fonts-master" to "pfUI-fonts" 4. Copy "pfUI-fonts" into Wow-Directory\Interface\AddOns 5. Restart Wow

Installation (The Burning Crusade, 2.4.3)

1. Download Latest Version 2. Unpack the Zip file 3. Rename the folder "pfUI-fonts-master" to "pfUI-fonts-tbc" 4. Copy "pfUI-fonts-tbc" into Wow-Directory\Interface\AddOns 5. Restart Wow
